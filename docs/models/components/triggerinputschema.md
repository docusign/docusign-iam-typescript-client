# TriggerInputSchema

Array of fields required as inputs to trigger the workflow, including field names, data types, and default values.

## Example Usage

```typescript

let value: TriggerInputSchema = {
  fieldName: "name",
  fieldDataType: "string",
  defaultValue: "",
};
```
